### Added
- Maintenance release - improve workflow, publishing, tidy ups, reuse compliance, deps.

(`expectGet()`, `expectPost()` etc).

- FlumExtension now works correctly with test class hierarchies.

## [1.1.0] - 2019-10-09
### Added
- Support for ignoring request order.
